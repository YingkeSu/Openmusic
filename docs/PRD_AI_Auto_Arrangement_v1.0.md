# Piano for AI 自动编曲系统 PRD（V1.0）

- 版本：V1.0（需求冻结版）
- 日期：2026-03-01
- 目标用户：内容创作者、音乐教育场景、独立音乐人

## 1. 目标

构建桌面本地优先的 AI 自动编曲产品，支持从自然语言意图到可编辑乐谱、可试听音频、可播放进度视频的完整闭环。

## 2. 需求冻结（MVP）

1. 单曲时长上限：60 秒。
2. 风格：古风优先，但用户可自定义风格。
3. 乐谱编辑：谱面拖拽优先。
4. 部署：桌面本地优先。
5. 导出：必须支持 `MusicXML + MIDI + MP4`。
6. 渲染：音频与视频必须本地渲染。

## 3. 核心功能

### FR-1 AI 自动编曲

1. 用户输入意图（风格、情绪、速度、调式、时长、难度）。
2. LLM 输出结构化编曲数据。
3. 编译生成简谱与五线谱（MusicXML）。

### FR-2 本地音频试听

1. 使用本地合成器将 MIDI 渲染为 WAV。
2. MVP 仅支持钢琴音色。

### FR-3 本地谱面进度视频

1. 根据乐谱与时间映射生成逐帧谱面。
2. 已播放区域黑色，未播放区域淡灰。
3. 本地编码输出 MP4。

### FR-4 手动微调

1. 支持谱面拖拽编辑音符。
2. 编辑后可一键重渲染音频与视频。

## 4. 非目标（MVP 不做）

1. 多乐器自动配器。
2. 实时多人协作编辑。
3. 云端渲染集群。

## 5. 成功指标

1. 可用性：用户可在 1 次流程内生成并导出三类文件。
2. 稳定性：有效输入下核心链路成功率 >= 95%。
3. 同步性：音画同步偏差 <= 80ms。
4. 编辑闭环：拖拽编辑后可重渲染且导出结果正确。

## 6. 风格策略说明

1. 风格影响生成策略层（提示词、模板、评估规则）。
2. 风格不影响底层渲染架构（MusicXML/MIDI 编译、本地音频、本地视频）。

## 7. 交付物

1. 桌面客户端（本地优先）。
2. 本地编排与渲染服务。
3. `MusicXML + MIDI + MP4` 导出。
4. PRD/SOP/SPEC/测试/发布文档。
