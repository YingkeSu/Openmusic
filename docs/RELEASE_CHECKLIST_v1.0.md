# 发布检查清单（Release Checklist V1.0）

## 1. 版本准备

1. PRD/SOP/SPEC 与实现一致。
2. 测试报告已归档。
3. 版本号与变更说明已更新。

## 2. 功能验收

1. 生成、编辑、重渲染、导出全流程通过。
2. 导出三件套完整。
3. 音画同步满足阈值。

## 3. 质量门禁

1. P0/P1 缺陷清零。
2. 性能测试结果达标。
3. 安全清单逐项确认。

## 4. 交付准备

1. 安装包可正常启动。
2. 用户使用文档可用。
3. 回滚方案可执行。

## 5. 发布结论

1. Go
2. Conditional Go
3. No-Go
