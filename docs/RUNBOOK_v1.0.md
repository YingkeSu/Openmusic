# 运维与排障手册（Runbook V1.0）

## 1. 常见故障

### 1.1 编曲失败

1. 查看 LLM 请求返回码。
2. 检查输入参数是否完整。
3. 重试生成（最多 3 次）。

### 1.2 音频渲染失败

1. 检查 MIDI 是否存在。
2. 检查 SoundFont 路径是否有效。
3. 验证 FluidSynth 可执行。

### 1.3 视频渲染失败

1. 检查 MusicXML 是否可解析。
2. 检查时间映射是否生成。
3. 验证 FFmpeg 参数与路径。

## 2. 诊断日志

1. 任务日志：`logs/tasks/*.log`
2. 渲染日志：`logs/render/*.log`
3. 导出日志：`logs/export/*.log`

## 3. 恢复步骤

1. 回滚到上一个可用版本。
2. 清理中间帧并重跑渲染。
3. 导出前做一致性校验。
