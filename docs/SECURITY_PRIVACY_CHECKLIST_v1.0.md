# 安全与隐私检查清单（V1.0）

## 1. 本地数据安全

1. 项目文件默认存储在用户目录，不上传云端。
2. 导出目录权限可控，避免全局可写。
3. 日志中不记录敏感凭证。

## 2. 依赖与执行安全

1. 固定关键依赖版本（music21、FFmpeg、FluidSynth）。
2. 外部命令执行参数做转义与白名单校验。
3. 禁止任意路径覆盖系统关键文件。

## 3. 输入安全

1. 参数校验（时长、路径、字符集、大小）。
2. 拖拽编辑事件数据结构校验。
3. 非法输入需返回可读错误，不崩溃。

## 4. 隐私与合规

1. 明确告知用户哪些数据会发往 LLM。
2. 提供本地项目删除与清理机制。
3. 导出文件不带内部调试信息。
