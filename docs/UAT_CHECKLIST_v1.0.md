# UAT 验收清单（V1.0）

## 1. 用户流程验收

1. 可从新建项目进入编曲界面。
2. 输入古风参数可生成可读乐谱。
3. 可试听音频且可播放进度视频。
4. 可拖拽编辑音符并触发重渲染。
5. 可导出 MusicXML、MIDI、MP4。

## 2. 结果正确性验收

1. 乐谱可在第三方工具打开。
2. MIDI 可在常见播放器打开。
3. MP4 可在常见播放器正常播放。
4. 音画同步感知无明显错位。

## 3. 稳定性验收

1. 连续使用 30 分钟无崩溃。
2. 关键流程失败时有明确错误提示。
3. 可恢复上一次版本继续编辑。

## 4. 最终结论

1. Go（通过）
2. Conditional Go（有条件通过）
3. No-Go（不通过）
